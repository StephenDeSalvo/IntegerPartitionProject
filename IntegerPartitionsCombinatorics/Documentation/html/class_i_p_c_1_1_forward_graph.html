Detailed Description

template<typename K, typename V>
class IPC::ForwardGraph< K, V >

Implementation of a forward graph.

Template Parameters
Kis the key value to look-up each vertex's data
His the hash value generated for K
Vis the type of data stored in each vertex

This is a bare-bones graph implementation, named "forward" because vertices and edges can only be added, not removed. Moreover, the graph is non-traversable in the classic sense: One can only iterate over its edges and vertices. This class does ensure that no vertices with the same key value exist, and it also ensures that there are no duplicate edges.
