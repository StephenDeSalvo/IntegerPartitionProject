Detailed Description

template<typename Z>
struct IPC::UniquePoint< Z >

A point in the integer plane with an ID.

Template Parameters
Zis the integer type used to assign the x and y coordinate values

IDs are not shared between points, hence they are "unique".

The compiler will check that Z is of integral type.
